# 图标文件说明

这个文件夹包含扩展的图标文件。

## 需要的图标尺寸
- icon16.png (16x16) - 扩展管理页面小图标
- icon32.png (32x32) - Windows系统图标
- icon48.png (48x48) - 扩展管理页面图标
- icon128.png (128x128) - Chrome Web Store图标

## 图标设计建议
- 使用简洁的设计
- 确保在小尺寸下清晰可见
- 使用与扩展功能相关的图标（如眼睛、隐藏符号等）
- 保持一致的颜色方案

## 临时解决方案
如果没有专业图标，可以：
1. 使用在线图标生成器
2. 从免费图标库下载
3. 使用简单的几何图形

推荐的图标主题：
- 眼睛图标（表示查看/隐藏）
- 减号或X符号（表示移除/隐藏）
- 盾牌图标（表示保护/过滤）
