# 弹窗尺寸选择指南

如果您遇到弹窗显示不全或被遮挡的问题，可以根据您的屏幕尺寸和浏览器环境选择合适的样式文件。

## 🎯 问题解决方案

### 方法1：修改HTML文件中的样式引用

打开 `popup.html` 文件，找到第7行：
```html
<link rel="stylesheet" href="popup-mini.css">
```

根据您的需要，将其替换为以下选项之一：

## 📏 可用的样式选项

### 1. 迷你版（推荐 - 解决显示不全问题）
```html
<link rel="stylesheet" href="popup-mini.css">
```
- **尺寸**: 300x400px
- **适用**: 小屏幕、低分辨率显示器
- **特点**: 最紧凑的布局，确保在任何环境下都能完整显示

### 2. 紧凑版
```html
<link rel="stylesheet" href="popup-compact.css">
```
- **尺寸**: 320x450px
- **适用**: 中等屏幕
- **特点**: 平衡了功能性和紧凑性

### 3. 标准版
```html
<link rel="stylesheet" href="popup.css">
```
- **尺寸**: 360x480px
- **适用**: 大屏幕、高分辨率显示器
- **特点**: 最舒适的视觉体验，但可能在小屏幕上显示不全

## 🔧 快速修复步骤

1. **确定问题**：如果弹窗被遮挡或显示不全
2. **选择样式**：推荐使用 `popup-mini.css`
3. **修改文件**：编辑 `popup.html` 第7行
4. **重新加载**：在浏览器扩展管理页面重新加载扩展
5. **测试效果**：点击扩展图标查看是否正常显示

## 📱 不同环境建议

### 笔记本电脑（13-15英寸）
- **推荐**: `popup-mini.css` 或 `popup-compact.css`
- **原因**: 屏幕空间有限，需要紧凑布局

### 台式机显示器（21英寸以上）
- **推荐**: `popup.css` 或 `popup-compact.css`
- **原因**: 屏幕空间充足，可以使用更舒适的布局

### 高DPI显示器
- **推荐**: `popup-mini.css`
- **原因**: 高DPI可能导致弹窗实际显示尺寸更大

### 浏览器缩放设置
- **缩放 > 100%**: 使用 `popup-mini.css`
- **缩放 = 100%**: 使用 `popup-compact.css`
- **缩放 < 100%**: 使用 `popup.css`

## 🎨 样式对比

| 特性 | 迷你版 | 紧凑版 | 标准版 |
|------|--------|--------|--------|
| 宽度 | 300px | 320px | 360px |
| 高度 | 400px | 450px | 480px |
| 字体大小 | 较小 | 中等 | 标准 |
| 间距 | 紧凑 | 适中 | 宽松 |
| 兼容性 | 最佳 | 良好 | 一般 |

## 🚀 自定义尺寸

如果预设的尺寸都不合适，您也可以：

1. **复制样式文件**：复制 `popup-mini.css` 为 `popup-custom.css`
2. **修改尺寸**：调整 `body` 中的 `width` 和 `height` 值
3. **应用样式**：在 `popup.html` 中引用您的自定义样式文件

### 自定义示例
```css
body {
    width: 280px;    /* 调整宽度 */
    height: 350px;   /* 调整高度 */
    /* 其他样式保持不变 */
}
```

## ⚠️ 注意事项

- 修改样式文件后需要重新加载扩展
- 过小的尺寸可能影响功能使用
- 建议先尝试预设的样式选项
- 如果仍有问题，可以进一步减小尺寸
